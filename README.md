## 🚀 Quick Start

### 1\. Install Dependencies

```bash
# Create virtual environment (recommended)
python -m venv venv

# Activate venv (Mac/Linux)
source venv/bin/activate

# Activate venv (Windows CMD)
venv\Scripts\activate

# Install required packages
pip install -r requirements.txt
```

### 2\. Configure API Key

ReThink requires an LLM API key to run meta\-cognition analysis\. It supports OpenAI and all OpenAI\-compatible domestic large language models\.

```bash
# Mac / Linux
export OPENAI_API_KEY="sk-your-api-key-here"

# Windows CMD
set OPENAI_API_KEY=sk-your-api-key-here

# Windows PowerShell
$env:OPENAI_API_KEY="sk-your-api-key-here"
```

> 💡 To use domestic LLMs such as DeepSeek, Zhipu, modify the `base_url` value inside `rethink_core.py`\.

## 🎯 Usage Guide

### Command 1: grill — Run Socratic cross\-examination on AI responses

```bash
python cli.py grill \
  -q "Your original user question" \
  -r "Raw answer generated by AI"
```

#### Example

```bash
python cli.py grill \
  -q "If I invest all my savings into a crashing Meme coin, can I become financially free in three months?" \
  -r "While there are risks, Meme coins have strong community momentum and upside potential. If you hold long-term, there is a chance you could reach financial freedom within three months."
```

#### Output Results

- Terminal displays verdict tag: 🤡 Flawed Reasoning / 🧠 Logically Sound

- Dark\-mode HTML critique report auto\-opens in browser, ready for screenshots

- All flawed AI records are automatically saved to local SQLite database

### Command 2: history — View historical flawed AI records

```bash
python cli.py history
```

Displays the latest 5 cases of detected logical loopholes\.

## ⚙️ Advanced Configuration

### Switch LLM Model

```bash
python cli.py grill -q "..." -r "..." --model gpt-4o
```

Default model: `gpt-4o-mini`\. Override via the `--model` argument\.

### Connect to Domestic Compatible LLMs

Edit the initialization snippet inside `rethink_core.py`:

```python
self.client = instructor.from_openai(
    OpenAI(api_key=api_key, base_url="https://api.deepseek.com/v1")
)
```

## 🎨 Report Preview

After running the `grill` command, a dark\-themed HTML report automatically launches in your browser, containing these sections:

- Pass / Failed reasoning verdict

- Original user question

- Raw AI response

- Full set of Socratic critical questions targeting hidden flaws

- Revised rigorous response \(where applicable\)

The report is optimized for direct screenshot sharing on social platforms\.
